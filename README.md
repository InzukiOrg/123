# Wellcome-TPU - модуль для Turbo Pascal
### Процедуры для вызова
* `wellcome('`**имя**`','`**номер**`');`
 * ***имя,номер - строковые данные***
### Подключение
* `uses wm;`
### Описание
* Легкая анимация владельца программы и номер лабараторки
* Генерируемые цвета
 * Light Blue(9)
 * Light Green(10)
 * Light Red(12)
 * Light Magenta(13)
 * Yellow(14)
